# India-DA-BA-Job-Market-Analysis
## India DA/BA Job Market Intelligence

### Problem
Freshers in India have no clear picture of what skills 
actually get you hired as a Data Analyst.

### What I did
Manually collected 200 live job postings from Naukri.com 
across Bangalore and Chennai. Built a SQLite database, 
wrote 6 SQL queries including window functions, and 
visualised findings in Excel.

### Key Findings
- Bangalore has 70.5% of all DA/BA openings
- Business Analysis is #1 demanded skill (32.5%)
- 55% of roles accept 0–1 year experience
- 98.5% of companies hide salary on Naukri
- MNCs dominate hiring at 47.5% share
- Accenture alone posted 61 openings (30.5%)

### Tools Used
Python, Pandas, SQLite, SQL Window Functions, 
Advanced Excel, Data Visualisation
